# Enterprise AI Use-Case Blueprints

Frameworks and templates for identifying, prioritising, and shaping high-value enterprise AI use cases.

## Why this exists

A lot of AI work inside organizations starts with tools instead of problems. That usually leads to scattered pilots, weak prioritisation, and unclear business value.

This repository is a working library of frameworks, scorecards, and planning templates for identifying AI opportunities that are useful, feasible, and worth implementing.

## What this repository will include

- AI use-case identification frameworks
- Prioritisation scorecards
- Business value vs. feasibility templates
- Risk and dependency mapping tools
- Use-case examples across teams and functions

## Current status

This repository is being built as a public working library. The first files will focus on use-case identification, prioritisation logic, and business-value framing.

## Working principle

Start with business problems, not model excitement.
